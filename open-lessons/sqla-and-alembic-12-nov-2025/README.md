# Таблицы


## Поставщик (Supplier)

### Поля
- id - bigint identity pk
- name - строка, можем ограничить 1000 символов. не пустое поле
- tax_number - ИНН. уникальный. строка. не пустой. ограничить по длине и значению (только цифры) 


## Тип Товара (ProductKind)

### Поля
- id - bigint identity pk
~~- slug - представление в виде slug для названия. уникальное поле~~
- name - строка, уникальное значение. можем ограничить 300 символов. не пустое поле


## Товар (Product)

### Поля
- id - bigint identity pk
- title - название. строка. не уник. ограничение до 1000 символов.
- description - описание. строка. не пустое.
- created_at - дата и время создания записи. не пустое, авто.
~~- updated_at - дата и время обновления записи. не пустое, авто.~~
~~- created_by - внешний ключ. автор записи~~
- kind_id - внешний ключ на тип товара
- supplier_id - внешний ключ на поставщика
